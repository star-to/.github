## 주요 기능
> 로그인 (카카오, 네이버 인증 : rest api 사용)
> 세션 사용으로 로그인 유지
> 로그아웃
- 현재 위치 지도 보기(카카오 map api 사용)
- 근처 음식점과 카페 리뷰 현황 보기 (별점, 상)
- 현재 위치로 부터 몇분 거리인지 구함
- 내가 리뷰했던 목록 보기
- 현재 위치가 카페 or 음식점일 경우 화장실 리뷰
